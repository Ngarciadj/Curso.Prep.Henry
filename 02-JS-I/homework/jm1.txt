VARIABLE var
Es una forma de guardar algo para poder usarlo 
despues, puede ser en forma de STRING, NUMBER 
o de BOOLEANOS

var a= 1

STRING Es una cadena de caracteres
puede ser una palabra, un numero; siempre 
expresado entre comillas.
Tipo de dato que expresa una variable

var "23"
var "hello"

NUMBERS (positivos o negativos), no lleva
comillas

BOOLEANOS expresan 2 opciones (true o false)

var a>5
var a=1
var true
